# Market Top Indicator Dashboard

Single-page, zero-build dashboard for GitHub Pages that tracks a weekly "Market Top Detector" strategy across 13 diversified global equity indexes.

## Deploy (GitHub Pages)

1. Push this repository to GitHub.
2. In **Settings → Pages**, set source to **Deploy from branch**.
3. Choose your default branch and `/ (root)`.
4. Save — the site will serve `index.html` directly.

No npm, bundlers, or frameworks are required.

## How data flows

The page loads data in this priority:

1. `data/signals.json` (preferred, generated by GitHub Actions)
2. `sessionStorage` cache for same-day browsing
3. Live Yahoo Finance fallback with endpoint + CORS proxy failover

## Strategy logic (weekly)

- EMA short = 10-week
- EMA long = 20-week
- MACD line = EMA12 - EMA26
- MACD signal = 9-week EMA of MACD
- RSI = 14-week Wilder smoothing
- ATR = 14-week Wilder smoothing

Signal states:
- **LONG**: EMA10 > EMA20 and MACD > signal
- **WATCH**: EMA10 > EMA20 and (MACD < signal or RSI warning)
- **FLAT**: EMA10 < EMA20

RSI warning is triggered when either:
- Bearish RSI divergence appears (price makes a higher high while RSI makes a lower high over the prior 10 bars)
- RSI momentum slope weakens from elevated levels (`RSI > 55` and 3-bar slope `< -6`)

Entry is recorded when state transitions from non-LONG to LONG.

## Automation

A weekly workflow runs Mondays at 09:00 UTC:

- File: `.github/workflows/weekly-refresh.yml`
- Script: `scripts/fetch_signals.py`
- Output: `data/signals.json`

## India market note

Yahoo Finance does not expose a direct NIFTY 500 ticker. This dashboard uses:

- `^NSEI` as **NIFTY 50 (India Broad Proxy)**
- `NIFTYSMLCAP250.NS` as a small-cap companion index

If Yahoo adds a direct NIFTY 500 symbol, it can be swapped in.
